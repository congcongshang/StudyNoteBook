# 跟谁学英语学习笔记

## 16种时态

| 时态 | 时间     | 特点     |
| ---- | -------- | -------- |
|      | 现在     | 一般     |
|      | 过去     | 完成     |
|      | 将来     | 进行     |
|      | 过去将来 | 完成进行 |

-   一般现在时: do/does
-   一般过去时: did
-   一般将来时: be going  to do
-   现在进行时: be doing

## 疑问句中文流程

**疑问词 + 时表词 + 主语 + 句剩 + 动词**

8个疑问词:  who/whom  (谁) what  whose  which  when  where  why  how 

每个时态的第一个词是**时表词**

每个时态的最后一个词是**动词**

时态剩下的就是句剩

例如: when are we going to eat?

what did you say to him?

By what way are you going to improve your English?
